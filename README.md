# list_of_lists
Alexa List of Lists skill
The List of Lists skill allows you to maintain multiple lists with Alexa.
To create a new list, say create list, and then the list name
To add an entry to your active list, say: add entry followed by the entry name
To switch to a different list say: change list to your list name
To delete a list, say: delete list, and the name of the list to delete
To remove an entry from a list, say: delete entry, followed by the entry name
To say all of your lists, say: what are my lists
To list the entries in your active list, say: what are my entries
